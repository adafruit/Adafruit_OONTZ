class  enc
 The enc class provides basic encoder support. It's polling-based (rather than interrupt-based) so it can work with any pins and any number of encoders, but the poll() function must be called frequently (a ~1 ms timer interrupt can optionally be used). Supports upper/lower limits with clipping or wraparound. Does not (currently) provide acceleration, detent division or shaft button debouncing...just basics, can use alternative libraries if special functionality is required. More...
